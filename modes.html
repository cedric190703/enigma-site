Voici les différentes commandes du Jeu :

    Le jeu possède 2 modes :

    Le mode Solo : Pour lancer une partie en mode solo.

    Cliquez sur solo dans le menu du jeu.

    Le mode multijoueur (maximum 5 joueurs): Pour lancer une partie en mode multijoueur,

    cliquez sur multijoueur dans le menu du jeu.

    Rentrez le nom de la salle à rejoindre ou à créer.

    Appuyez sur créer si vous souhaitez créer la salle

    pour vous et vos amis ou appuyez et vos amis ou appuyez sur rejoindre

    si la salle existe déjà et que vous souhaitez la rejoindre.
